# web-lab3 Королькова Елизавета

В рамках данной работы я разработала веб приложение игры 2048.

В работе был использован чистый javascript + html и css

Логика игры
1. Сетка 4x4 генерируется динамически с помощью JavaScript. В начале игры на поле есть 1-3 случайных ячейки с значением 2 либо 4
   (картинка)
   (картинка)
   (картинка)
3. После каждого хода корректно появляется 1-2 новых плитки с значением 2 или 4.
   (картинка)
4. Слияние плиток однаковых чисел работает во всех направлениях

Взаимодействие вне игрового поля
1. Реализована проверка на окончание игры (Нет возможных ходов)
2. Кнопка "Начать заново" сбрасывает состояние поля и счет (Без перезагрузки страницы)
3. Реализована кнопка "Отмена хода"
4. В таблице отражается топ-10 рекордов, которые сохранил пользователь.

Взаимодействие с игрой
1. Поддержка управления кнопками клавиатуры для десктопа и виртуальными кнопками на мобильных устройствах
   (картинка)
2. Состояние игры сохраняется в localStorage

Визуальное отображение
1.Каждое значение плитки имеет свой цвет
(картинка)
2. Реализована плавная анимация движения и слияния плиток
